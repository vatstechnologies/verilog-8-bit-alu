# verilog-8-bit-alu
8 - bit ALU design using verilog with testbench and waveform in simulation
##Overview
This Project Implements an 8 bit ALU (Arithmetic Logic Unit) using Verilog HDL.

##Objectives
* Design an 8 Bit ALU
* Perform arithmetic and logic operations
* Verifying Functionality using a testbench
* Simulate waveforms results

## Tools Used 
*Verilog HDL
*Icarus Verilog
*EDA Playground
